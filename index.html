IOT Solar PV Home Automation Project

I am designing and building an automated solar panel that rotates to the optimal angle based off of light detected from the sun using photoresistors. This Project uses a arduino board to calculate and send signals to servo motors to rotate based off of C++ code. The aim of the project is to reach maximum output from a single solar panel by experimenting with solar PVs to determine if the overall net energy output increases from automating solar PVs. Documents and photos will be uploaded soon.

Fusion Energy Fuel Research Project

For this project I am researching and addressing the issues of inertial confinement fusion fuel pellets. Aiming to find solutions to the cost and quality control issues of the current manufacturing methods this project is currently underway and will conclude in December 2023

Underwater Autonomous Vehicle Control Simulation (UAVCS)

UAVCS is a python program written with the help of Dr. James Myatt for the Dean's Research Award. This project is the creation a control simulation for a 6-degree of freedom underwater vehicle, the project focuses on the scopes of control theory, python programming and underwater vehicle dynamics. From the physics of an underwater vehicle I created graphical simulations by solving a system of differential equations to determine the depth of the vehicle for varying conditions.
