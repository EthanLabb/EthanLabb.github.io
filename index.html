Underwater Autonomous Vehicle Control System (UAVCS)

UAVCS is a python program written with the help of Dr. James Myatt for the Dean's Research Award. This project is the creation a control simulation for a 6-degree of freedom underwater vehicle, the project focuses on the scopes of control theory, python programming and underwater vehicle dynamics. From the physics of an underwater vehicle I created graphical simulations by solving a system of differential equations to determine the depth of the vehicle for varying conditions.
